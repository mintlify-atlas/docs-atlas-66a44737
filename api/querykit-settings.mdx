## Overview

The `QueryKitSettings` class provides a fluent API for configuring QueryKit's behavior. It allows you to customize operator syntax, define property mappings, configure case sensitivity, and set parsing options.

This class is typically used within a configuration action passed to `QueryKitConfiguration` or when using the `ApplyQueryKit` extension methods.

**Namespace:** `QueryKit.Configuration`

## Methods

### Property\<TModel\>(Expression\<Func\<TModel, object\>\>? propertySelector)

Creates a property mapping for a specific property on a model.

**Parameters:**
- `propertySelector` - Expression selecting the property to map

**Returns:** `QueryKitPropertyMapping<TModel>` for fluent configuration

```csharp
settings.Property<User>(u => u.FirstName)
    .HasQueryName("name")
    .PreventSort();
```

### DerivedProperty\<TModel\>(Expression\<Func\<TModel, object\>\>? propertySelector)

Creates a derived property mapping for computed or custom properties.

**Parameters:**
- `propertySelector` - Expression defining the derived property logic

**Returns:** `QueryKitPropertyMapping<TModel>` for fluent configuration

```csharp
settings.DerivedProperty<User>(u => u.FirstName + " " + u.LastName)
    .HasQueryName("fullName");
```

### CustomOperation\<TModel\>(Expression\<Func\<TModel, ComparisonOperator, object, bool\>\> operationExpression)

Creates a custom operation mapping for complex filtering logic.

**Parameters:**
- `operationExpression` - Expression defining the custom operation logic

**Returns:** `QueryKitCustomOperationMapping<TModel>` for fluent configuration

```csharp
settings.CustomOperation<User>((user, op, value) => 
    user.Tags.Any(t => t.Name == value.ToString()))
    .HasName("hasTag");
```

## Usage Examples

### Basic Configuration

```csharp
using QueryKit.Configuration;

var config = new QueryKitConfiguration(settings =>
{
    // Customize operators
    settings.EqualsOperator = "eq";
    settings.ContainsOperator = "contains";
    
    // Enable flexible filtering
    settings.AllowUnknownProperties = true;
});
```

### Property Mappings

```csharp
var config = new QueryKitConfiguration(settings =>
{
    // Alias property names
    settings.Property<Product>(p => p.Name)
        .HasQueryName("productName");
    
    // Prevent sorting on sensitive fields
    settings.Property<User>(u => u.PasswordHash)
        .PreventSort()
        .PreventFilter();
    
    // Create derived properties
    settings.DerivedProperty<Order>(o => o.Items.Sum(i => i.Price))
        .HasQueryName("totalPrice");
});
```

### Database-Specific Features

```csharp
var config = new QueryKitConfiguration(settings =>
{
    // Enable phonetic search with PostgreSQL
    settings.DbContextType = typeof(MyDbContext);
    
    // Use sounds-like operator
    settings.SoundsLikeOperator = "sounds";
});

// Query: "Name sounds 'Jon'" matches "John", "Jon", etc.
var results = dbContext.Users
    .ApplyQueryKitFilter("Name sounds 'Jon'", config)
    .ToList();
```

### Case-Insensitive Comparisons

```csharp
var config = new QueryKitConfiguration(settings =>
{
    settings.CaseInsensitiveAppendix = "i";
    settings.CaseInsensitiveComparison = CaseInsensitiveMode.Upper;
});

// Query: "Name ==i 'john'" matches "John", "JOHN", "john"
var results = dbContext.Users
    .ApplyQueryKitFilter("Name ==i 'john'", config)
    .ToList();
```
